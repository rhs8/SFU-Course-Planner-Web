# [SFU Course Planner](https://rhs8.github.io/SFU-Course-Planner-Web/)

SFU Course Planner is a mobile app designed to help students efficiently plan and track their academic progress. It centralizes course information, requirements, and scheduling into one cohesive platform, reducing dependence on external resources and manual tracking.

It :

     - Lets students browse and select courses with detailed lecture and lab information
     
     - Visualizes progress toward degree completion through clear, interactive layouts
     
     - Integrates real-time course data, schedules, and instructor information
     
     - Offers a cohesive calendar with key academic dates and personalized reminders

     - Supports long-term, multi-semester planning with prerequisite validation

     - Follows a clean Material Design interface built with Kotlin and XML
